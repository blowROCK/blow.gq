안드로이드에서 “당겨서 새로고침(Pull to Refresh)" UI를 간단히 구현해보자.

당겨서 새로고침 기능을 구현해주는 ```SwipeRefreshLayout```은 Android support library v4에 포함되어 있다.
먼저 gradle에 support v4 라이브러리를 추가한다.

```gradle
// build.gradle

dependencies {
  ...
  compile 'com.android.support:support-v4:23.4.0'
}
```
